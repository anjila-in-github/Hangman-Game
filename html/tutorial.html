☞Hangman is played by guessing the letters of a word.
☞Player is given different categories to choose.
☞After choosing certain category, the gaming page is opened.
☞Once start button is clicked, the game begins.
☞Player is given 5 lives.
☞Each time a letter is guessed incorrectly, player's life decreases.
☞Player has to guess the word within a time limit.
☞To make the game easier, hint is also provided.

☛If the timer ends before the word is guessed then the game ends.
☛If the 5 lives of player is finished then the game ends
